=============
Cropper (Web)
=============

Getting started
===============

- Install `dart 2.x`_
- install development server and builder:
  .. code-block::
     
     pub global activate webdev

- Install project dependencies:
  .. code-block::

     pub get

- Run development server:
  .. code-block::
     
     webdev serve

- Go to http://localhost:8080


.. _`dart 2.x`: https://webdev.dartlang.org/tools/sdk#install
